Jutzi 2008 1D P-alpha Compaction Wave Validation

Overview

This case reproduces the 1D porous aluminum compaction wave test from Jutzi et al. (2008) Section 5.1, comparing the following physical quantities at a fixed time against the paper’s SPH curves and the analytical solution:

  • Pressure distribution
  • Distension alpha distribution

This type of test is used to examine the performance of the p-alpha model in terms of wave front structure, compaction zone thickness, and post-wave compaction degree.

This run used shock_00350.h5 as the reference frame. Extracted results:

  • Material 0 particle count: 400
  • Pressure range: ~1.07e-01 Pa to 5.07e+09 Pa
  • Distension range: 1.0 to 1.275

The compaction wave has driven the material from the initial porous state to a nearly fully compacted region.

Validation Objective

The physical picture of this case: a right-side moving piston drives a compaction wave through an initially porous aluminum rod. As the wave propagates forward, the material transitions from an initial distension of

$$ \alpha_0 = 1.275 $$

toward a denser state. Unlike pure dense-material impact cases, attention must be paid not only to pressure magnitude but also to how pore collapse unfolds spatially.

Thus, this test clearly exposes the spatial structure characteristics of the p-alpha model:

  • Whether the wave front position is accurate
  • Whether the compaction transition zone is reasonable
  • Whether the post-wave alpha falls to the correct level
  • Whether the pressure curve matches the reference in both magnitude and shape

If the model roughly matches the overall trend but shows significant deviations in wave front structure, compaction zone thickness, or post-wave state, this case will directly expose those issues.

How to Run

Test directory:

1

GASPHiA-Tests/Jutzi2008_Section5_1_PAlphaCompactionWave

Run with:

1

./run_all.sh --source-dir /path/to/GASPHiA --cuda 0

The default workflow executes sequentially:

  1. Compile and run input/gen_input_1d.cpp
  2. Compile GASPHiA using the local para.cuh
  3. Run compaction_wave.ini
  4. Read output/shock_00350.h5
  5. Output pressure / distension comparison plots

Note: this case’s para.cuh reserves a macro specifically for reproducing the experimental path:

1

#define JUTZI2008_5_1_reproduce_need 1

Result Interpretation

The figure is divided into two panels:

  • Top: Pressure profile
  • Bottom: Distension alpha profile

Each panel simultaneously plots three curves:

  • GASPHiA computed results
  • Paper SPH curve
  • Analytical reference curve

For this type of case, the evaluation emphasis is not pixel-level coincidence but structural consistency at the following levels:

  1. Whether the wave front appears near the same position
  2. Whether the spatial extents of the post-wave high-pressure zone and compaction zone are similar
  3. Whether alpha transitions smoothly from the initial 1.275 to approximately 1.0
  4. Whether the overall profile shape is consistent with the literature trend

From the result plot of this run, the above features are all clearly identifiable, indicating that the p-alpha chain is valid on this classic 1D test.

Reference Value for Future Users

When modifying the p-alpha model, porosity state variable update, pressure coupling, or porous material input parameters in GASPHiA, this test should be treated as mandatory.

Its advantage lies in direct feedback and rapid problem localization:

  • Wave front curve offset → possible wave speed or boundary driving anomaly
  • alpha unable to compress to target value post-wave → possible compaction model malfunction
  • Pressure plateau too high or too low → possible EOS–porosity model coupling deviation
  • Transition zone thickness significantly off → possible influence of numerical dissipation, kernel function, or artificial viscosity

For model tuning, such validation cases — where problem areas can be quickly located through profile plots — often hold greater value than tests relying solely on final integral quantities.

Summary

Jutzi2008_Section5_1_PAlphaCompactionWave is a typical spatial profile validation case for porous materials. Unlike single-point Hugoniot tests that only provide a single state, this case fully unfolds the spatial structure of the compaction wave, making it more suitable for verifying whether the core logic of the p-alpha model is correct.

The actual run results show:

  • Both pressure and alpha profiles form reasonable comparisons with the literature reference
  • The compaction wave drives the material from alpha = 1.275 to near alpha = 1.0
  • The overall structure of the wave front, compaction zone, and post-wave state is credible

For users solving porous material problems with GASPHiA, this case should be treated as a standard validation item for confirming model correctness, not an optional auxiliary test.

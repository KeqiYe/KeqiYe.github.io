Jutzi 2008 一维 p-alpha 压实波验证

概述

本算例复现 Jutzi et al. (2008) 第 5.1 节的一维多孔铝压实波测试，在固定时刻对比以下物理量在 GASPHiA、论文 SPH 曲线与解析解之间的一致性：

  • 压力分布
  • 膨胀度 alpha 分布

该类测试用于考察 p-alpha 模型在波前结构、压实区厚度、波后压实程度方面的表现。

本次运行以 shock_00350.h5 作为对照帧。提取结果：

  • 材料 0 粒子数：400
  • 压力范围：约 1.07e-01 Pa ~ 5.07e+09 Pa
  • 膨胀度范围：1.0 ~ 1.275

压实波已将材料从初始多孔态推进至近乎完全压实的区域。

验证目标

本算例的物理图景如下：右侧移动活塞推动压实波穿过一根初始存在孔隙的多孔铝柱。随着波向前传播，材料从初始膨胀度

$$ \alpha_0 = 1.275 $$

逐渐向更致密状态过渡。与纯致密材料冲击算例不同，此处不仅需关注压力幅值，还需关注孔隙塌缩如何随空间展开。

因此，该测试能清晰暴露 p-alpha 模型的空间结构特性：

  • 波前位置是否准确
  • 压实过渡带是否合理
  • 波后 alpha 是否回落至正确水平
  • 压力曲线是否与参考结果在量级和形态上一致

若模型仅总趋势大致吻合，但波前结构、压实区厚度或波后状态存在明显偏移，该算例能直接暴露这些问题。

运行方式

测试目录：

1

GASPHiA-Tests/Jutzi2008_Section5_1_PAlphaCompactionWave

执行以下命令启动：

1

./run_all.sh --source-dir /path/to/GASPHiA --cuda 0

默认流程依次执行：

  1. 编译并运行 input/gen_input_1d.cpp
  2. 使用当前目录下的 para.cuh 编译 GASPHiA
  3. 运行 compaction_wave.ini
  4. 读取 output/shock_00350.h5
  5. 输出压力 / 膨胀度对比图

需额外注意：本算例的 para.cuh 中预留了一个用于复现实验路径的宏

1

#define JUTZI2008_5_1_reproduce_need 1

结果解读

图形分为上下两部分：

  • 上图：压力剖面
  • 下图：膨胀度 alpha 剖面

每部分均同时绘制三条曲线：

  • GASPHiA 计算结果
  • 论文 SPH 曲线
  • 解析参考曲线

对该类算例，评估重点并非逐点像素级重合，而是以下层面的结构一致性：

  1. 波头是否出现在相同位置附近
  2. 波后高压区与压实区的空间范围是否相近
  3. alpha 是否从初始 1.275 平滑过渡至约 1.0
  4. 剖面整体形状是否与文献结果趋势一致

从本次运行结果图来看，上述特征均可清晰辨认，表明 p-alpha 链路在该经典一维测试上是成立的。

实际运行数据

后处理提取的关键指标如下：

指标数值
对照帧shock_00350.h5
材料 0 粒子数400
压力最小值1.07e-01 Pa
压力最大值5.07e+09 Pa
膨胀度最小值1.0000
膨胀度最大值1.2750

这些数值从两个角度印证了模拟的有效性：

一方面，输出中的 alpha 已覆盖从初始孔隙态到完全压实态的整个范围；另一方面，压力峰值已进入 GPa 量级，说明压实波强度足够，已形成真实压实结构，而非在弱扰动状态下进行无效比较。

对后续用户的参考价值

若需在 GASPHiA 中修改 p-alpha 模型、孔隙状态量更新、压力耦合或多孔材料输入参数，该测试应作为必测项。

其优势在于反馈直接、问题定位快：

  • 波前曲线偏移 → 可能为波速或边界驱动异常
  • alpha 波后无法压至目标值 → 可能为压实模型工作不正常
  • 压力平台偏高或偏低 → 可能为 EOS 与孔隙模型耦合存在偏差
  • 过渡带厚度明显偏离 → 可能为数值耗散、核函数或人工粘性影响过大

对模型调参而言，此类通过剖面图即可快速定位问题区域的验证算例，其价值往往高于仅依赖最终积分量的测试。

小结

Jutzi2008_Section5_1_PAlphaCompactionWave 是一个典型的多孔材料空间剖面验证算例。与单点 Hugoniot 仅给出单点状态不同，该算例将压实波的空间结构完整展开，更适合用于检验 p-alpha 模型的核心逻辑是否正确。

实际运行结果显示：

  • 压力剖面与 alpha 剖面均与文献参考形成合理对照
  • 压实波将材料从 alpha = 1.275 推进至接近 alpha = 1.0
  • 波前、压实区和波后状态的整体结构可信

对于使用 GASPHiA 求解多孔材料问题的用户，该算例应作为确认模型正确性的标准验证项，而非可选的辅助测试。
